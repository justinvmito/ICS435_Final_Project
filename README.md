# ICS435 Final Project
This is a multi-layer perceptron model used to predict a plot of land's rice/paddy yield, given various predictors, in a Jupyter Notebook environment.
<br>
It was trained on the [Paddy Dataset](https://archive.ics.uci.edu/dataset/1186/paddy+dataset) by Muthukumaran Subramaniyan, from UC Irvine's Machine Learning Repository.
